## windows 查看 ip 地址信息

1. 键入 `ipconig` 查看。
2. 找到 `无线局域网适配器 WLAN` 信息
3. 然后虚拟机配置
   - 同一个网段的 ip 地址
   - 相同的子网掩码
   - 相同的网关信息（不一样会导致虚拟机无法联网）

测试能否联网：

```bash
ping www.baidu.com
```



## 修改 ip 地址

```bash
vi /etc/sysconfig/network-scripts/ifcfg-ens33
```

修改完成后 `:wq` 报存并退出。

### 重启网关

```bash
ifdown ens33
#
ifup ens33
```

然后，输入 `ip addr` 查看本机 ip 地址信息。
